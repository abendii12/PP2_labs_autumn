mkdir - помогает создать файлы внутри файлов 
cd - заходить в определенную файлу по команде 
git init - инициализирует нашу дерикторию в репрезиторию git add - добавляет страницу в репрезиторию
git commit -m "description of your project" - делает комкит нашего дериктории объясняя последнию изменению git status - проверяет текущею состоянию репрезитории он дает информацию сохранилось ли текущая состояния репрезитории git restore - отменяет измении
git log- можно посмотреть все коммиты и измении которые произоали
git tag nameTag - делает тэги для коммитов 
git push - отправляет в репрезитории нашу дерикторию
